সিএসএস(CSS) :

এইচটিএমল ডকুমেন্টের যেকোন এলিমেন্টকে স্টাইলিং বা একটা রুপ দিতে সিএসএস ব্যবহার হয়। একটা প্যারাগ্রাফ (p) বা হেডিং (h1) বা যেকোন এলিমেন্ট কে ধরুন রং করতে চান, ফন্ট বড় ছোট করতে হবে, অবস্থান এক দিক থেকে অন্যদিকে নিতে হবে, ব্যাকগ্রাউন্ড রং বদলাতে হবে এরুপ শত ধরনের স্টাইল পরিবর্তন সিএসএস দিয়ে করা হয়। বিশেষ করে লেআউট তৈরীর জন্য সিএসএস সবচেয়ে বেশি জরুরি।

বর্তমানে সিএসএস ৩ চলে এসেছে (এখনও কাজ চলছে)। এর আগের ভার্সন হল সিএসএস ২.১। তবে সিএসএস ৩ এখন অধিকাংশ ব্রাউজারে সাপোর্ট করে এবং CSS 3 ব্যবহার দিন দিন বেড়েই চলছে।

CSS কি? What is CSS?

CSS যার অর্থ হলো Cascading Style Sheets CSS এবং এটি একটা design language। মানে website design সম্পর্কিত কাজ করার জন্য আমরা এই CSS ব্যবহার করে থাকি। এর মাধ্যমে আমরা আমাদের website এর কোন HTML document এর style নির্ধারণ করে থাকি। যেমন- কোন একটা বক্স দেখতে কেমন হবে, তার রং কি রকম হবে, এর দৈর্ঘ্য বা প্রস্থ কত হবে ইত্যাদি। এমনকি আমরা CSS এর মাধ্যমে বলে দেই যে যত প্রকার HTML tag রয়েছে তা আমাদের ওয়েবসাইটে কি রকম দেখাবে। আমরা যখন কোন ওয়েবসাইট তৈরী করি তখন প্রথমে markup অথবা HTML কোড লিখে থাকি। কিন্তু এই HTML কোডগুলোতে যদি কোন style দেওয়া না হয় তাহলে HTML ফাইলটি পরিপুর্ণতা পায় না বা আমাদের ডিজাইন মত হয় না বা যেভাবে HTML কোডগুলো লেখা হয়েছে সেভাবেই দেখাবে। কিন্তু আমরা যদি HTML ফাইলকে আমাদের ডিজাইন মত দেখাতে চাই তাহলে আমাদেরকে অবশ্যই CSS ব্যবহার করতে হবে। যার ফলে আমরা কোন ট্যাগকে ইচ্ছা মত কালার, ফন্ট সাইজ ইত্যাদি নির্ধারণ করে দিতে পারব।

আমরা বলতে পারব যে কোন অংশটুকু কতটুকু জায়গা নিবে বা কোন দিকে এই অংশ টুকু দেখাবে ইত্যাদি কাজ করার জন্য আমরা CSS ব্যবহার করতে পারব। যেমন- মনে করুন আমরা একটি বাড়ি বানাতে চাই। তার জন্য আমাদের ইট, বালি, সিমন্টে, রড লাগবে। এই ইট, বালি, সিমেন্ট, রড ইত্যাদি হলো HTML ট্যাগ যার ফলে আমাদের বাড়ির কাঠামো তৈরী করতে পারি। কিন্তু আমরা যদি চাই বাড়ির এই দিকে এই রং হবে, দেওয়ালে একটা ছবি থাকবে, ছবির চারদিকে একটা বর্ডার থাকবে, পুরো বাড়ির একপাশ আকাশী কালারের হবে ইত্যাদি হলো CSS এর মূল কাজ।

CSS কত প্রকার? :

CSS কত প্রকার বলতে বুঝানো হচ্ছে যে অমরা কয়ভাবে CSS Code লিখতে পারব। CSS Code আমরা

  • ১। Inline CSS
  • ২। Internal CSS
  • ৩। External CSS

১। Inline CSS

Inline CSS Code লিখার জন্য আমরা HTML ট্যাগের ভিতরে style নামের একটি এট্রিবিউট নিয়ে তার মধ্যে CSS কোড লিখে থাকি।

২। Internal CSS

Internal CSS Code লিখতে হলে আমাদেরকে অবশ্যই HTML File এর head tag এর ভিতরে style নামে tag টি নিয়ে CSS Code লিখতে হবে।

৩। External CSS

Inline CSS এবং Internal CSS Code যেভাবে লিখি External CSS Code ও সেভাবে লিখতে হবে তবে শুধুমাত্র পার্থক্য হলো External CSS Code লিখতে হলে আপনাকে নতুন একটি CSS File তৈরী করতে হবে। এই CSS File এর ভিতরে আপনাকে CSS Code লিখতে হবে।

CSS ইতিহাস :

10 অক্টোবর 1994 সালে Håkon Wium Lie দ্বারা CSS প্রথম প্রস্তাব করা হয় । [২৪] একই সময়ে ওয়েবের জন্য বেশ কিছু অন্যান্য স্টাইল শীট ভাষা প্রস্তাব করা হয়েছিল, এবং পাবলিক মেইলিং তালিকা এবং ওয়ার্ল্ড ওয়াইড ওয়েব কনসোর্টিয়ামের অভ্যন্তরে আলোচনার ফলে প্রথম W3C CSS সুপারিশ (CSS1) [25] প্রকাশিত হয়েছিল 1996 সালে। বিশেষ করে , বার্ট বসের একটি প্রস্তাব প্রভাবশালী ছিল; তিনি CSS1-এর সহ-লেখক হন, এবং CSS-এর সহ-স্রষ্টা হিসেবে বিবেচিত হন। [২৬]

1980 এর দশকে স্ট্যান্ডার্ড জেনারেলাইজড মার্কআপ ল্যাঙ্গুয়েজ ( এসজিএমএল ) শুরু হওয়ার পর থেকে স্টাইল শীটগুলি এক বা অন্য আকারে বিদ্যমান ছিল এবং ওয়েবের জন্য স্টাইল শীট সরবরাহ করার জন্য CSS তৈরি করা হয়েছিল। [২৭] ওয়েব স্টাইল শীট ভাষার জন্য একটি প্রয়োজনীয়তা ছিল স্টাইল শীটগুলি ওয়েবে বিভিন্ন উত্স থেকে আসা। অতএব, DSSSL এবং FOSI এর মতো বিদ্যমান স্টাইল শীট ভাষাগুলি উপযুক্ত ছিল

এইচটিএমএল বাড়ার সাথে সাথে এটি ওয়েব ডেভেলপারদের চাহিদা মেটাতে বিভিন্ন ধরনের শৈলীগত ক্ষমতাকে অন্তর্ভুক্ত করে । এই বিবর্তনটি ডিজাইনারকে আরও জটিল HTML এর খরচে সাইটের চেহারার উপর আরো নিয়ন্ত্রণ দিয়েছে। ওয়েব ব্রাউজার বাস্তবায়নের বিভিন্নতা , যেমন ViolaWWW এবং WorldWideWeb , [28] সামঞ্জস্যপূর্ণ সাইটের উপস্থিতি কঠিন করে তোলে, এবং ব্যবহারকারীরা কীভাবে ওয়েব বিষয়বস্তু প্রদর্শিত হয় তার উপর কম নিয়ন্ত্রণ রাখেন। টিম বার্নার্স-লি দ্বারা বিকাশিত ব্রাউজার/সম্পাদকের স্টাইল শীট ছিল যা প্রোগ্রামে হার্ড-কোড করা ছিল। তাই স্টাইল শীটগুলি ওয়েবে নথিগুলির সাথে লিঙ্ক করা যায়নি৷ [২৪] CERN-এর রবার্ট ক্যালিয়াউও প্রেজেন্টেশন থেকে কাঠামোটিকে আলাদা করতে চেয়েছিলেন যাতে বিভিন্ন স্টাইল শীট মুদ্রণ, স্ক্রিন-ভিত্তিক উপস্থাপনা এবং সম্পাদকদের জন্য বিভিন্ন উপস্থাপনা বর্ণনা করতে পারে। [২৮]

এখানে 20টি CSS প্রশ্ন ও উত্তর বাংলায় অনুবাদ করা হয়েছে :

1. CSS এর পূর্ণরূপ কি?

উত্তরঃ স্ক্যালিং স্টাইল শিট (Cascading Style Sheets)।

2. CSS কি?

উত্তরঃ CSS হলো ওয়েব পেজের ডিজাইন, লেআউট এবং স্টাইলিং করার জন্য ব্যবহৃত একটি মার্কআপ ভাষা।

3. CSS এর কোন পূর্ণরূপ হলো?

উত্তরঃ স্ক্যালিং স্টাইল শিট।

4. CSS এর মুল কাজ কি?

উত্তরঃ CSS এর মুল কাজ হলো ওয়েব পেজের ডিজাইন এবং লেআউট করা।

5. একটি HTML এলিমেন্টের স্টাইল কোন উপায়ে প্রযোজ্য করা যায়?

উত্তরঃ HTML এলিমেন্টের স্টাইল প্রযোজ্য করার উপায় হলো CSS বা ইনলাইন স্টাইল, ইন্টারনাল স্টাইল এবং এক্সটার্নাল স্টাইল।

6. স্টাইল শিট কী?

উত্তরঃ স্টাইল শিট হলো একটি নথি বা ফাইল যা ওয়েব পেজের উপর ডিজাইন এবং স্টাইলিং প্রভাব ফেলে।

7. CSS কোন ধরণের সিলেক্টর সম্পর্কে আলোচনা করে?

উত্তরঃ বিভিন্ন ধরণের সিলেক্টর যেমন - এলিমেন্ট সিলেক্টর, ক্লাস সিলেক্টর, আইডি সিলেক্টর, ইমোজি সিলেক্টর, অ্যাট্রিবিউট সিলেক্টর, প্যাসেন্ট সিলেক্টর, সাইবিলিটি সিলেক্টর, চাইল্ড সিলেক্টর, কম্বাইনেটর সিলেক্টর ইত্যাদি।

8. বোর্ডার সিলেক্টর কীভাবে কাজ করে?

উত্তরঃ বোর্ডার সিলেক্টর ব্যবহার করে আপনি একটি এলিমেন্টের বর্ডার প্রপার্টিগুলি সেট করতে পারেন।

9. কীভাবে CSS এর মাধ্যমে এলিমেন্টের স্টাইলিং করা যায়?

উত্তরঃ আপনি এলিমেন্টের স্টাইল প্রয়োজনীয় সম্পর্কিত প্রপার্টিগুলি সেট করে স্টাইল প্রয়োগ করতে পারেন।

10. আমি কীভাবে এক্সটার্নাল স্টাইলিং করতে পারি?

উত্তরঃ আপনি একটি স্টাইল ফাইল তৈরি করে এবং তারপরে তাকে HTML ফাইলের link ট্যাগের মাধ্যমে সংযোজন করতে পারেন।

11. স্টাইল শিট লাঙ্গুয়েজে প্রয়োগ করার সময় যেসব ইভেন্টগুলি ব্যবহার করা যায়, তা কি?

উত্তরঃ স্টাইল শিট লাঙ্গুয়েজে যেসব ইভেন্টগুলি ব্যবহার করা যায় তা হলো: hover, focus, active, visited, link, nth-child, nth-of-type, first-child, last-child, before, after ইত্যাদি।

12. ফ্লেক্সবক্স কি?

উত্তরঃ ফ্লেক্সবক্স হলো একটি স্থানান্তরের মডেল যা কন্টেন্টগুলি সারানোর জন্য ব্যবহৃত হয়। এটি একটি অভ্যন্তরীণ কন্টেন্ট স্পেস থেকে বাইরের কন্টেন্টের সাথে সাথে আপডেট করে।

13. কীভাবে গ্রিড কন্টেন্ট লেআউট তৈরি করতে পারি?

উত্তরঃ আপনি CSS গ্রিড প্রপার্টিগুলি ব্যবহার করে গ্রিড কন্টেন্ট লেআউট তৈরি করতে পারেন।

14. পজিশনিং প্রপার্টির বিভিন্ন মান কী?

উত্তরঃ পজিশনিং প্রপার্টির বিভিন্ন মান হলো - static, relative, absolute, fixed, sticky।

15. জাভাস্ক্রিপ্ট ব্যবহার করে কিভাবে স্টাইল প্রোপার্টিগুলি চেঞ্জ করা যায়?

উত্তরঃ জাভাস্ক্রিপ্ট ব্যবহার করে স্টাইল প্রোপার্টিগুলি চেঞ্জ করতে হলে, ডম এলিমেন্টের স্টাইল প্রোপার্টিগুলি এক্সেস করে এবং তার মান চেঞ্জ করতে হবে।

16. রেস্পন্সিভ ওয়েব ডিজাইন কি?

উত্তরঃ রেস্পন্সিভ ওয়েব ডিজাইন হলো একটি ওয়েব ডিজাইনের ধারণা যা ডিভাইসের আকার, অনুমতি এবং স্ক্রিনের আকারের উপর ভিত্তি করে ওয়েব পেজের লেআউট এবং ডিজাইন সহজে সম্পর্কেত হয়।

17. আমি কীভাবে মিডিয়া কুয়েরি ব্যবহার করতে পারি?

উত্তরঃ মিডিয়া কুয়েরি প্রপার্টিগুলি ব্যবহার করে, ডিভাইসের প্রস্তুতিতে ভিত্তি করে ওয়েব পেজের লেআউট ও ডিজাইন চেঞ্জ করা যায়।

18. কীভাবে আমি একটি স্যাসপেন্ড মিডিয়া কুয়েরি লেখতে পারি?

উত্তরঃ আপনি স্যাসপেন্ড মিডিয়া কুয়েরি ব্যবহার করে ডিভাইসের আকার, অনুমতি এবং স্ক্রিনের আকারের উপর ভিত্তি করে ওয়েব পেজের লেআউট ও ডিজাইন চেঞ্জ করতে পারেন তবে এটি মিডিয়া সংলগ্নভাবে স্থায়ী থাকবে যখন ব্যবহারকারী পাতা স্ক্রোল করবেন।

19. কীভাবে ইনলাইন স্টাইলিং করতে পারি?

উত্তরঃ HTML এলিমেন্টের স্টাইল সেট করতে আপনার ইনলাইন স্টাইল অ্যাট্রিবিউট ব্যবহার করতে হবে।

20. কীভাবে গ্রিড কন্টেন্ট লেআউট তৈরি করতে পারি?

উত্তরঃ আপনি CSS গ্রিড প্রপার্টিগুলি ব্যবহার করে গ্রিড কন্টেন্ট লেআউট তৈরি করতে পারেন।
